# Terminal BlackJack

How to?

    1. Clone git repository
    
`git clone https://github.com/jdc-linux/tbj`

    2. Change to tbj directory

`cd tbj`

    3. Copy to bin folder or make executable where it is

`cp tbj /bin`

    4. Make file executable

`sudo chmod +x /bin/tfm`
